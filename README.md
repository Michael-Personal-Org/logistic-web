## Stack

| Categoría | Tecnología |
|-----------|------------|
| Framework | Next.js 16 (App Router) |
| Lenguaje | TypeScript 5 |
| Estilos | Tailwind CSS 4 + shadcn/ui |
| Estado | Zustand |
| Data fetching | TanStack Query + Axios |
| Formularios | React Hook Form + Zod |
| Linting | Biome |

## Setup local

```bash
bun install
cp .env.example .env.local
bun run dev
```
